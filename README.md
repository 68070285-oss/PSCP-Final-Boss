# KmitlChat – Simple Real-time Web Chat

แชทเรียลไทม์ด้วย Node.js + Express + Socket.IO

## Features
- ตั้งชื่อก่อนเข้าใช้งาน
- ข้อความเรียลไทม์
- แจ้งเตือนเข้า/ออกห้อง
- บันทึกประวัติแชทลง `messages.json`

## Project Structure
/public
    index.html
    welcome.html
    client.js
    style.css
server.js
package.json
messages.json (auto)

```bash
npm install
npm run dev
